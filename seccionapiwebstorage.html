API de almacenamiento web HTML

HTML Web storage, es mejor que las cookies!

¿Qué es el almacenamiento web ?

Con el almacenamiento web, las aplicaciones web pueden almacenar datos localmente dentro del navegador del usuario.

Antes de HTML5, los datos de las aplicaciones debían almacenarse en cookies, incluidas en cada solicitud del servidor.
El almacenamiento web es más seguro y se pueden almacenar grandes cantidades de datos localmente, sin afectar el rendimiento del sitio web.

A diferencia de las cookies, el límite de almacenamiento es mucho mayor (al menos 5 MB)
y la información nunca se transfiere al servidor.

El almacenamiento web es por origen (por dominio y protocolo).
Todas las páginas, desde un mismo origen, pueden almacenar y acceder a los mismos datos.

Objetos de almacenamiento web HTML

El almacenamiento web HTML proporciona dos objetos para almacenar datos en el cliente:

  • window.locaStorage- almacena datos sin fecha de vencimiento.
  • window.sessionStorage- almacena datos para una sesión (los datos se pierden cuando se cierra la pestaña del navegador)

Antes de usar el almacenamiento web, verifique la compatibilidad del navegador con localStorage y sessionStorage:

if (typeof(Storage) !== "undefined") {
  // Code for localStorage/sessionStorage.
 } else {
  // Sorry! No Web Storage support..
 }

localStorage

El objeto localStorage almacena los datos sin fecha de vencimiento.
Los datos no se eliminarán al cerrar el navegador y estarán disponibles al día, semana o año siguiente.

Ejemplo

// Store
localStorage.setItem("lastname", "Prodan");

// Retrieve
 document.getElementById("result").innerHTML = localStorage.getItem("lastname");

Ejemplo explicado:

  • Cree un par de localStorage con nombre= "lastname" y valor= "Prodan"
  • Recupere el valor de "lastname" e insértelo en el elemento con id= "result"

El ejemplo anterios también podría escribirse así:

// Store
localStorage.lastname = "Smith";
// Retrieve
 document.getElementById("result").innerHTML = localStorage.lastname;

La sintaxis para eliminar el elemento localStorage "lastname" es la siguiente:

localStorage.removeItem("lastname");

Nota: Los pares de nombre/valor (name/value) siempre se almacenan como cadenas.
¡Recuerda convertirlos a otro formato cuando sea necesario!

El siguiente ejemplo cuenta la cantidad de veces que un usuario ha hecho click en un botón.
En este código la cadena de valor se convierte a un número para poder aumentar el contador:

Clickea el bóton para ver el contador aumentar.

Cierre la pestaña (o ventana) del navegador y vuelva a intentarlo, y el contador seguirá contando (no se restablecerá).

if (localStorage.clickcount) {
  localStorage.clickcount = Number(localStorage.clickcount) + 1;
} else {
  localStorage.clickcount = 1;
}
document.getElementById("result").innerHTML = "Haz clickeado el bóton" +
localStorage.clickcount + "vez(veces).";

sessionStorage

El objeto sessionStorage es igual al objeto localStorage, excepto que almacena los datos para una sola sesión.
Los datos se eliminan cuando el usuario cierra la pestaña especifica del navegador.

El siguiente ejemplo cuenta la cantidad de veces que un usuario ha hecho click en un botón en la sesión actual:

Ejemplo

Clickea el bóton para ver el contador aumentar.

Cierre la pestaña (o ventana) del navegador, vuelva a intentarlo y el contador se restablecerá.

if (sessionStorage.clickcount) {
  sessionStorage.clickcount = Number(sessionStorage.clickcount) + 1;
} else {
  sessionStorage.clickcount = 1;
}
document.getElementById("result").innerHTML = "Haz clickeado el bóton " +
sessionStorage.clickcount + " vez(veces) en esta sesión.";
