Elementos del código de programación

HTML contiene varios elementos para definir la entrada del susuario y el código de programación.

<kbd> para entrada de teclado

El elemento HTML <kbd> se utiliza para mostrar la entrada del teclado. El contenido interior
se muestra en la fuente monoespaciada predeterminada del navegador.

Ejemplo

Defina algo de texto como entrada de teclado en un documento:

         <p> Guarde el documento presionando <kbd>Ctrl + S<kbd> </p>

Resultado:

Guarde el documento presionando Ctrl + S

<samp> Para el Output

El elemento HTML <samp> se utiliza para mostrar un ejemplo Output de un codigo de programación.
El contenido interior se muestra en la fuente monoespaciada predeterminada del navegador.

"La etiqueta HTML <samp> se utiliza para mostrar ejemplos de salida de programas o
fragmentos de código en una página web. Proporciona una forma de destacar y formatear
este texto de salida, lo que lo hace más fácil de identificar en el contexto de la página."

Ejemplo

Define algo de texto como una muestra Output de un codigo de programación.

<p>Mensaje desde mi computadora:</p> <p><samp>File not found.<br>Press F1 to continue</samp></p>

Resultado:

Mensaje desde mi computadora:

File not found.
Press F1 to continue

<code> Para codigos de programación

El elemento HTML <code> se utiliza para definir una parte de un codigo de programación.
El contenido interior se muestra en la fuente monoespaciada predeterminada del navegador.

"La etiqueta HTML <code> se utiliza para resaltar y formatear fragmentos de código en una página web.
Al utilizar esta etiqueta, el texto contenido dentro de <code> se muestra en una fuente de ancho fijo,
como una fuente de máquina de escribir, y suele estar rodeado por un borde o fondo distintivo para diferenciarlo del texto normal.
Esto es especialmente útil cuando deseas mostrar ejemplos de código fuente, comandos de programación
o cualquier otro contenido relacionado con la codificación en tu página web.
La etiqueta <code> ayuda a mantener la estructura y la legibilidad del código dentro del contexto de tu página."

Ejemplo:

Define algo de codigo de programación como texto en un documento:

          <code> x = 5; y = 6; z = x + y; </code

Resultado:

x = 5; y = 6; z = x + y;

Note como el elemento <code> no preserva espacios en blanco extra y saltos de linea.
Para arreglar esto, puedes poner el <code> adentro de un elemento <pre>

          <pre>           <code> x = 5; y = 6; z = x + y; </code           </pre>

Resultado:

x = 5;
y = 6;
z = x + y;

"La etiqueta <pre> en HTML se utiliza para definir texto preformateado,
Lo que significa que el texto dentro de esta etiqueta se muestra tal como está escrita en el código fuente,
incluyendo espacion en blanco y saltos de linea. Esta etiqueta es útil cuando deseas mostrar código fuete,
bloques de texto con un formato específico, o cualquier contenido donde la disposición exacta de los caracteres es importante. "

<var> Para variables

El elemento HTML <var> se utiliza para definir una variable en programación o en una Expresión matemática.
Su contenido normalmente se muestra en cursiva.

Ejemplo

Defina algo de texto como variables en un documento:

<p>
   El area de un triangulo es : 1/2 x <var>b</var x <var>h</var, donde <var>b</var> es la base,
   y <var>h</var> es la altura vertical.
</p>

Resultado:

El area de un triangulo es : 1/2 x b x h, donde b es la base, y h es la altura vertical.

"<var> se utiliza para representar una variable o una expresión matemática en un contexto de texto.
Esta etiqueta se utiliza para indicar que un fragmento de texto dentro de una página web representa una variable o una expresión matemática.
Por lo general, el texto dentro de <var> se muestra en cursiva para distinguirlo del texto circundante.

Es útil en situaciones donde deseas resaltar que ciertos elementos en tu texto son variables, como en explicaciones matemáticas o cientificas.

Resumen del capítulo

  • <kbd> define una entrada de teclado
  • <samp> define una salida de muestra de un programa de computadora
  • <code> define un fragmento de código de programación
  • <var> define una variable en programación o en una expresión matemática
  • <pre> define texto preformateado
