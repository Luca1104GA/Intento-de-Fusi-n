Links CSS

Los links HTML son hipervínculos.
Los links se encuentran en casi todas las páginas web.
Los links permiten a los usuarios hacer clic para pasar de una página a otra.
Cuando mueves el mouse sobre un link, la flecha del mouse se convertirá en una manita.
La etiqueta HTML <a> define un hipervínculo.

Nota: Un link no tiene que ser texto. ¡Un link puede ser una imagen o cualquier otro elemento HTML!

El atributo más importante del elemento <a> es el atributo href, que indica el destino del link.
El texto del link es la parte que será visible para el lector.
Al hacer clic en el texto del link, se enviará al lector a la dirección URL especificada.

De forma predeterminada, los Links aparecerán de la siguiente manera en todos los navegadores:

Un link no visitado está subrayado y en azul.
Un link visitado está subrayado y en color violeta.
Un link activo está subrayado y en rojo.

Consejo: Por supuesto, los links se pueden diseñar con CSS para darle otro aspecto.

Atributos de Destino

De forma predeterminada, la página vinculada se mostrará en la ventana actual del navegador.
Para cambiar esto, debe especificar otro destino para el enlace.
El atributo target especifica dónde abrir el documento vinculado.
puede tener uno de los siguientes valores:

  • _self- Por defecto. Abre el documento en la misma ventana/pestaña en la que se hizo clic.
  • _blank- Abre el documento en una nueva ventana o pestaña.
  • _parent- Abre el documento en el marco principal.
  • _top- Abre el documento en el cuerpo completo de la ventana.

Use mailto: dentro del atributo href
para crear un link que abra el programa de correo electrónico del usuario
(para permitirle enviar un nuevo correo electrónico)

<a href="mailto:someone@example.com">Send email</a>

Imagen como Link

Para usar una imagen como link, simplemente coloque la etiqueta <img> dentro de la etiqueta <a>

<a href="https://www.ejemplo.com"> <img src="imagen.jpg" alt="Descripción de la imagen">

Botón como Link

Para utilizar un botón HTML como link, debe agregar algún código JavaScript.
JavaScript le permite especificar qué sucede en ciertos eventos, como hacer clic en un botón:

El siguiente boton lo creamos de esta manera
Primero dentro del body creamos una division con el id"botoncontenedor"
a este le damos una funcion de boton con el id "botontop" para despues personalizarlos en CSS.
Luego hay que darle codigo JavaScript y eso lo hacemos utilizando el elemento <script> dentro del body.

<script>
const botonTop = document.getElementById("botonTop");
botonTop.addEventListener("click", () => {
window.scrollTo({ top: 0, behavior: "smooth" });
}); </script>

1. const botonTop = document.getElementById("botonTop"); :Esto busca el elemento del botón con el ID "botonTop" en el documento HTML y lo almacena en la variable
2. botonTop. Esto te permite interactuar con el botón utilizando JavaScript.
3. botonTop.addEventListener("click", () => { ... }); :Esto agrega un escuchador de eventos al botón. En este caso, estamos escuchando el evento "click", lo que significa que cuando el botón se hace clic, se ejecutará el código que se encuentra dentro de la función de flecha () => { ... }
4. window.scrollTo({ top: 0, behavior: "smooth" }); :Dentro de la función de flecha, estamos utilizando el método scrollTo en el objeto window para desplazar la ventana hacia la parte superior de la página.
{ top: 0, behavior: "smooth" } son opciones que se pasan al método. top: 0 significa que queremos desplazarnos hasta la posición vertical 0, es decir, la parte superior de la página.
behavior: "smooth agrega un efecto de desplazamiento suave en lugar de un salto abrupto.

Un Link también puede tener el estilo de un botón mediante CSS:

<style> a:link, a:visited { background-color: #f44336; color: white; padding: 15px 25px; text-align: center; text-decoration: none; display: inline-block; } a:hover, a:active { background-color: red; } </style>

  • a:link a:visited : Estilos para enlaces antes y despues de ser visitados, Cambia el color y subrayado del texto para resaltar los enlaces.
  • background-color: Color de fondo
  • padding: Espacio interno alrededor del contenido. Ajusta la distancia entre el contenido y el borde.
  • text-align: Controla la alineacion horizontal del texto.
  • text-decoration: Modifica la apariencia de la decoración del texto, como subrayar, tachar, resaltar o quitar la decoración
  • display: Controla como se muestra un elemento. (en linea bloque o de otra manera )
  • a:hover: Cambia el estilo cuando el cursor del mouse se coloca sobre él. Proporciona retroalimentación visual al usuario antes de hacer clic en un enlace.
  • a:active: Estilo que se aplica cuando un enlace está siendo activamente presionado (haciendo clic) por el usuario. Puede ser útil para mostrar el momento en que se hace clic.

Títulos de enlaces

El atributo title especifica información adicional sobre un elemento.
La información suele mostrarse como texto de información sobre herramientas
cuando el ratón se mueve sobre el elemento.

Crear Marcadores

Los links se pueden utilizar para crear marcadores,
de modo que los lectores puedan saltar a partes específicas de una página web.
Los marcadores pueden resultar útiles si una página web es muy larga.
Para crear un marcador, primero cree el marcador y luego agréguele un enlace.
Cuando se hace clic en el enlace, la página se desplazará hacia abajo o hacia arriba hasta la ubicación con el marcador.

Primero use el atributo id para crear un marcador.

<h2 id="C4">Capítulo 4</h2>

Luego, agregue un link al marcador ("Saltar al Capítulo 4"), desde la misma página:

<a href="#C4">Saltar al Capítulo 4</a>
