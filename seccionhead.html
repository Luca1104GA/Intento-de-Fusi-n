El elemento "Head"

El elemento HTML <head> es un contenedor para los siguientes elementos : <title>, <style>
<meta>, <link>, <script> y <base>.

<head>

Este elemento es un contenedor de metadatos (datos sobre datos) y se coloca entre
la etiqueta <html> y la etiqueta <body>

Los metadatos HTML son datos sobre el documento HTML. Los metadatos no se muestran.
Los metadatos suelen definir el titulo del documento, el set de caracteres, los estilos,
los scripts y otra metainformación.

<title>

El elemento <title> define el título del documento. El título debe ser solo texto
y se muestra en la barra del navegador o en la pestaña de la página.

El <title> es obligatorio en docuemntos HTML!

El contenido del título de una página es muy importante para la optimización de motores de búsqueda (SEO*)
Los algoritmos de los motores de búsqueda utilizan el título de la página para decidir el orden
al enumerar las páginas en los resultados de búsqueda.

"(SEO)" se refiere a "Search Engine Optimization" o "Optimización para Motores de Búsqueda" en español.
El SEO es una estrategia y conjunto de técnicas utilizadas para mejorar la visibilidad de un sitio web
en los resultados de los motores de búsqueda como Google, Bing y otros. El objetivo del SEO es hacer
que un sitio web aparezca en los primeros resultados de búsqueda cuando los usuarios buscan palabras clave relacionadas con su contenido.

El elemento <title>

  • define un título en la barra de herramientas del navegados
  • proporcina un título para la página cuando se agrega a favoritos
  • muestra un título para la página en los resultados del motor de búsqueda

Por lo tanto, intenta que el título sea lo más preciso y significativo posible!

Ejemplo

<!DOCTYPE html> <html> <head> <title>Un title de página significativo</title> </head> <body> El contenido del documento...... </(body> </html>

<style>

El elemento <style> se utiliza para definir información de estilo para una sola página HTML:

ejemplo

<style> body {background-color: powderblue;} h1 {color: red;} p {color: blue; } </style>

<link>

El elemento <link> define la relación entre el documento actual y un recurso externo.
se usa comúnmente para vincular a hojas de estilo externas.

Ejemplo:

<link rel="stylesheet href="mystyle.css">

<meta>

El elmento <meta> se utiliza normalmente para especificar el juego de caracteres, descripción de una pagina,
palabras claves, autor del documento, y configuración de la ventana gráfica.

Los metadatos no seran mostrados en la pagina, pero es utilizado por los navegadores (como mostrar contenido o recargar la pagina)
, por motores de búsqueda (palabras claves), y otros servicios web.

Ejemplos

Define el juego de carateres utilizados (character set)

<meta charset="UTF-8">

Define palabras claves para los motores de búsqueda

<meta name="keywords" content="HTML, CSS, javaScript">

Define una descripcion de tu pagina web

<meta name="description" content="Tutorial gratis en Linea">

Define el autor de la Pagina

<meta name="author" content="Luca Prodan">

Actualizar documento cada 30 segundos

<meta http-equiv="refresh" content="30">

Configura la ventana gráfica para hacer que tu pagina web se vea bien en todos los dispositivos

<meta name="viewport" content="width=device-width, initial-scaled=1.0">

Toda informacion "meta" va dentro de la seccion "head"

Configurando la ventana gráfica (Viewport)

La ventana gráfica (viewport) es el area visible para el usuario de la página web. Varía según el dispositivo -
Sera más pequeña en un teléfono celular que en la pantalla de una computadora.

Deberias incluir los siguientes <meta> en todas tus páginas Web:

<meta name="viewport" content="width=device-width, initial-scaled=1.0">

Esto le da al navegador instrucciones de como controlar las dimensiones y escalas de la pagina.

El width=device-width establece el ancho de la página para que siga el ancho del dispositivo
(que variara dependiendo el dispositivo)

initial-scale=1.0 Esto significa que la página se mostrará inicialmente a su tamaño real, sin zoom.

<script>

El elemento <script> se utiliza para para definir javaScript al lado del cliente.

La etiqueta HTML <script> se utiliza para incorporar código de secuencias de comandos.
(generalmente javaScript) en una página web. Esta secuencia de comandos, conocida como
"cliente", se ejecuta en el navegador web del usuario, permitiendo la interacción dinámica y
el procesamiento de datos en la página. A través del elemento <script>, los desarroladores
web pueden agregar funcionalidades personalizadas y mejoras a sus sitios web. lo que les
permite crear experiencias más interacivas y dinámicas para los visitantes.

Para ver mas acerca de javaScript visite la seccion javaScript

<base>

El elemento <base> especifica la URL base y/o el objetivo para todas las URL relativas en la página.
esta etiqueta debe tener presente un atributo href, target presente o ambos.
Solo puede haber un elemento <base> en un documento!

El elemento <base> es una etiqueta HTML utilizada para especificar la URL y/o el
objetivo para todas las URL relativas dentro de una página. Esto significa que podemos
establecer una URL base única y luego referenciar recursos como Imágenes, hojas de estilo
y enlaces relativos a partir de esa base.

Resumen del capítulo

  • <head> es un contenedor de metadatos (datos sobre datos)
  • <head> se ubica entre las etiquetas <html> y <head>
  • <title> es obligatorio y define el title de un documento
  • <style> se utiliza para definir información de estilo para un documento
  • <link> es usado comunmente para vincular a hojas de estilo externas
  • <script> se utiliza para definir javaScript del lado del cliente
  • <base> especifica la URL base y/o el destino de todas las URL relativas de una página
  • <meta> especifica el juego de caracteres, la descripción de
    la página, las palabras clave, el autor del documento y la configuracion de la ventana gráfica
