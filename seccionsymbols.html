Simbolos HTML

Símbolos o letras que no están presentes en tu teclado pueden ser agragados a HTML usando entidades.

Simbolos de entidades HTML

Muchos símbolos matemáticos, técnicos y monetarios no están presentes en un teclado normal.
Para agregar dichos símbolos a una página HTML, puede usar el nombre de la entidad o un número de la
entidad (una referencia decimal o hexadecimal) para el símbolo.

Mostrar el signo del Euro:
<p>Se mostrará &euro;</p>
<p>Se mostrará &#8364;</p>
<p>Se mostrará &#x20AC;</p>
Se verá como:
Se mostrará €
Se mostrará €
Se mostrará €
