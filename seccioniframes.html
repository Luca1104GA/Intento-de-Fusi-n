Iframes

El término "iframe" se traduce al español como "marco en línea" o "marco flotante".
Un iframe es un elemento HTML que permite incrustar otro documento HTML dentro de una página web.
Se utiliza para mostrar contenido de otras fuentes, como sitios web externos, documentos o elementos multimedia,
en una ventana o marco dentro de una página web.

La idea es que el contenido del iframe se muestre dentro de un espacio delimitado en la página principal.

Sintaxis de marco en línea

La etiqueta <iframe> especifica un marco en línea.
Un marco en línea se utiliza para incrustar otro documento dentro del documento actual.

Sintaxis

<iframe src="url" title="descripcion"></iframe>

Consejo: Es una buena práctica incluir siempre un atributo title para el archivo <iframe>.
Los lectores de pantalla lo utlizan para leer cuál es el contenido del iframe.

Iframe - Establecer ancho y altura

Utilice los atributos height width para especificar el tamaño del iframe.
La altura y el ancho se especifican en píxeles de forma predeterminada.

Ejemplo:

<iframe src="demo_iframe.htm" height="200" width="300" title="iframe Example"></iframe>

O puedes agregar el atributo style y usar propiedades CSS height width

<iframe src="demo_iframe.htm"style="height:200px;width:300px;" title="iframe Example"></iframe>

Eliminar el borde del iframe

De forma predeterminada, un iframe tiene un borde alrededor.
Para eliminar el borde, agregue el atributo style y use la proiedad CSS border:

<iframe src="demo_iframe.htm"style="border:none;" title="iframe Example"></iframe>

Con CSS, también puedes cambiar el tamaño, estilo y color del borde del iframe:

<iframe src="demo_iframe.htm"style="border:2px solid red;" title="iframe Example"></iframe>

Destino de un enlace iframe

Se puede utilizar un iframe como marco de destino para un enlace.
El atributo del enlace target debe hacer referencia al atributo name del iframe:

Ejemplo:

El Rey a Hablado

<iframe src="demo_inframe.htm" name="iframe_a" title="Iframe Example></iframe> <p> <a href="https://www.youtube.com" target="iframe_a">El Rey a Hablado</a> </p>

Resumen del capítulo

  • La etiqueta HTML <iframe> especifica un marco en línea.
  • El atributo src define la URL de la página a incrustar.
  • Incluya siempre un atributo title (para lectores de pantalla)
  • Los atributos height y width especifica el tamaño del iframe.
  • Utilice border:none; para eliminar el borde alrededor del iframe.
