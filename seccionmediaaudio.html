Audio HTML

El elemento <audio> se utiliza para reproducir un archivo de audio en una página web.

Ejemplo

Para reproducir un archivo de audio en HTML, use el elemento <audio>

<audio controls>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>

¿Cómo funciona?

El atributo controls agrega controles de audio, como reproducción pausa y volumen.

El elemento <source> le permite especificar archivos de audio alternativos entre los que el navegador puede elegir.
El navegador utilizará el primer formato reconocido.

El texto entre la etiqueta de <audio> solo se mostrará en navegadores que no admitan el elemento <audio>.

Autoplay

Para iniciar un archivo de audio automáticamante use el atributo autoplay.

<audio controls autoplay>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>

Nota: Los navegadores Chromium no permiten la reproducción automática en la mayoría de los casos.
Sin embargo, siempre se permite la reproducción automática silenciada.

Muted

Agreuge muted despúes del autoplay para permitir que su archivo de audio comience a reproducirse automáticamente (pero silenciado):

<audio controls autoplay muted>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>

Metodos, Propiedades y eventos.

El DOM HTML define métodos, propiedades y eventos para el elemento <audio>

Esto le permite cargar, reproducir y pausar audios, así como configurar la duración y el volumen.

También hay eventos DOM que pueden notificarle cuando un audio comienza a reproducirse, se pausa, etc.

Para obtener una referencia DOM completa, vaya a Referencia DOM de audio/video HTML.
