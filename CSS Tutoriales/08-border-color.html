Color del Borde CSS

AnteriorSiguiente

La propiedad border-color se utiliza para establecer el color de los cuatro bordes.

El color se puede configurar mediante:

  • Name : especifique un nombre de color, como "red"
  • HEX : especifique un valor HEX, como "#ff0000"
  • RGB : especifique un valor RGB, como "rgb(255,0,0)"
  • HSL : especifique un valor HSL, como "hsl(0.100%. 50%)"
  • Transparente

Nota: Si el border-color no se establece, hereda el color del elemento.

ejemplo

Demostración de los diferentes colores de borde:

p.one {
  border-style: solid;
  border-color: red;
}

p.two {
  border-style: solid;
  border-color: green;
}

p.three {
  border-style: dotted;
  border-color: blue;
}

Colores laterales específicos

La propiedad border-color puede tener de uno a cuatro valores
(para el borde superior, el borde derecho, el borde inferior y el borde izquierdo).

ejemplo

p.one {
  border-style: solid;
  border-color: red green blue yellow; /* Rojo arriba, Verde a la derecha, Azul abajo y Amarillo a la izquierda */
}
Rojo arriba, Verde a la derecha, Azul abajo y Amarillo a la izquierda.

Valores HEX

El color del borde también se puede especificar mediante un valor hexadecimal (HEX):

ejemplo

p.one {
  border-style: solid;
  border-color: #ff0000; /* rojo */
}
Rojo.

Valores RGB

O usando valores RGB:

ejemplo

p.one {
  border-style: solid;
  border-color: rgb(255, 0, 0); /* rojo */
}
Rojo.

Valores HSL

Tmbién puede utilizar valores HSL:

ejemplo

p.one {
  border-style: solid;
  border-color: hsl(0, 100%, 50%); /* rojo */
}
Rojo.

Puede obtener más información sobre los valores HEX, RGB y HSL en nuestros capítulos de Colores CSS
