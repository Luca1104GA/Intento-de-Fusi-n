Comentarios CSS

AnteriorSiguiente

Los comentarios CSS no se muestran en el navegador, pero pueden ayudar a documentar su código fuente.

Los comentarios se utilizan para explicar el código y pueden ayudar cuando edite el código fuente en una fecha posterior.

Los navegadores ignoran los comentarios.

Un comentario CSS se coloca dentro del elemento <style>, comienza /* y termina con */

Ejemplo

/* Este es un comentario de una sola línea */
p {
  color: red;
}

Puedes agregar comentarios donde quieras en el código:

p {
  color: red;  /* Establece el color del texto en rojo */
}

Incluso en medio de una línea de código:

p {
  color: /*Rojo*/red; 
}

Los comentarios también pueden abarcar varias líneas:

/* Esto es
un comentario
con multiples lineas */

p {
  color: red;
}

Comentarios HTML Y CSS

En el tutorial de HTML, aprendiste que puedes agregar comentarios a tu fuente HTML usando la sintaxis <!--...-->

En el siguiente ejemplo, utilizaremos una combinación de comentarios HTML y CSS:

Ejemplo

<!DOCTYPE html>
<html>
<head>
<style>
p {
  color: red; /* Establece el color del texto en rojo */
}
</style>
</head>
<body>

<h2>Mi título</h2>

<!-- Estos párrafos estarán en rojo -->
<p>¡Hola Mundo!</p>
<p>Este párrafo está diseñado con CSS.</p>
<p>Los comentarios HTML y CSS no se muestran en el resultado.</p>

</body>
</html>
