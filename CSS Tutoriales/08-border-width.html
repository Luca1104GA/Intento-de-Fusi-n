Ancho del Borde

AnteriorSiguiente

La propiedad border-width especifica el ancho de los cuatro bordes.

El ancho se puede establecer como un tamaño específico (in px, pt, cm, em, etc)
o utilizando uno de los tres vamores predefinidos: thin, medium, o thick:

ejemplo

Demostracion de los diferentes border-width:

p.one {
  border-style: solid;
  border-width: 5px;
}

p.two {
  border-style: solid;
  border-width: medium;
}

p.three {
  border-style: dotted;
  border-width: 2px;
}

p.four {
  border-style: dotted;
  border-width: thick;
}

Resultado:

5px border-width
medium border-width
2px border-width
thick border-width

Anchuras Específicas de los Lados

La propiedad border-width puede tener de uno a cuatro valores
(para el borde superior, el borde derecho, el borde inferior y el borde izquierdo):

ejemplo

p.one {
  border-style: solid;
  border-width: 5px 20px; /* 5px arriba y abajo, 20 px en los lados. */
}

p.two {
  border-style: solid;
  border-width: 20px 5px; /* 20px arriba y abajo, 5px en los lados. */
}

p.three {
  border-style: solid;
  border-width: 25px 10px 4px 35px; /* 25px arriba, 10px a la derecha, 4px abajo y 35px a la izquierda. */
}
5px arriba y abajo, 20 px en los lados.
20px arriba y abajo, 5px en los lados.
25px arriba, 10px a la derecha, 4px abajo y 35px a la izquierda.
