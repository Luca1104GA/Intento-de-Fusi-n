CSS Border - Shorthand Property

Como vimos en la página anterior, hay muchas propiedades a considerar cuando se trata de los bordes.

Para acortar el código, támbien es posible especificar todas las propiedades de borde individuales en una sola propiedad.

La propiedad border es una propiedad abreviada para las siguientes propiedades de borde individuales:

  • border-width
  • border-style(requerido)
  • border-color

ejemplo

p {
  border: 5px solid red;
}

Algo de Texto

También puede especificar todas las propiedades de borde individuales para un solo lado:

Borde izquierdo

p {
  border-left: 6px solid red;
}
