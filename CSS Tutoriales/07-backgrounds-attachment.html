CSS Background Attachment

AnteriorSiguiente

La propiedad background-attachment especifíca si la imagen de fonde debe desplazarse o ser fija (no se desplazara con el resto de la página):

Ejemplo

Especifique que la imagen de fondo debe arreglarse:

body {
  background-image: url("img_tree.png");
  background-repeat: no-repeat;
  background-position: right top;
  background-attachment: fixed;
}

Ejemplo

Especifique que la imagen de fondo debe desplazarse con el resto de la página:

body {
  background-image: url("img_tree.png");
  background-repeat: no-repeat;
  background-position: right top;
  background-attachment: scroll;
}
