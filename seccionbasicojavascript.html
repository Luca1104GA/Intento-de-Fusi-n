javaScript

JavaScript hace que las páginas HTML sean más dinámicas e interactivas.

La etiqueta HTML <script>

La etiqueta HTML <script> se utiliza para incorporar código de secuencias de comandos.
(generalmente javaScript) en una página web. Esta secuencia de comandos, conocida como
"cliente", se ejecuta en el navegador web del usuario, permitiendo la interacción dinámica y
el procesamiento de datos en la página. A través del elemento <script>, los desarroladores
web pueden agregar funcionalidades personalizadas y mejoras a sus sitios web. lo que les
permite crear experiencias más interacivas y dinámicas para los visitantes.

La etiqueta HTML <javaScript> se utiliza para definir un script en el lado del cliente (javaScript).
Este elemento contiene instruciones de script o apunta a un archivo de script externo a través del atributo src.
Los usos comunes de javaScript son la manipulación de imágenes, la validación de formulario y los cambios dinámicos de contenido.

Una muestra de javaScript

Para seleccionar un elemento UTML, javaScript utiliza con mayor frecuencia el método document.getElementById.

Ejemplos:

Este ejemplo de javaScript escribe "¡Hola javaScript!" en un elemento HTML con id="demo" :

javaScript puede cambiar el contenido de un Elemento

Esto es una demostracion!.

document.getElementById("demo").innerHTML = "¡Hola javaScript!";

javaScript puede cambiar estilos:

javaScript puede cambiar el estilo de un elemento HTML.

document.getElementById("demo").style.fontSize = "25px"; document.getElementById("demo").style.color = "red"; document.getElementById("demo").style.backgroundColor = "yellow";

javaScript puede cambiar atributos:

Aca un javaScript cambie el valor del atributo src de la imagen.

document.getElementById("image").src = "picture.gif";

La etiqueta <noscript>

Esta etiqueta define un contenido alternativo que se mostrará a los usuarios que tienen scripts desahabilitados
en su navegador o tienen un navegador que no admite scripts.

<script> document.getElementById("demo").innerHTML = "¡Hola javaScript!"; </script> <noscript>Perdon, tu navegador no soporta javaScript!</noscript> <!--Un navegador sin soporte javaScript mostrara el texto escrito dentro de <noscript> ––>
