Listas

Las listas permiten a los desarrolladores web agrupar un conjunto de elementos relacionados en listas.

Ejemplo:
    Desordenada
  • Artículo
  • Artículo
  • Artículo
  • Artículo
    Ordenada
  1. Primer elemento
  2. Segundo artículo
  3. Tercer elemento
  4. Cuarto elemento

Lista desordenada

Una lista desordenada comienza con la etiqueta <ul>. Cada elemento de la lista comienza con la etiqueta <li>.
Los elementos de la lista estarán marcados con viñetas (pequeños círculos negros) de forma predeterminada:

Ejemplo:

<ul> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ul>

Lista ordenada

Una lista ordenada comienza con la etiqueta <ol>. Cada elemento de la lista comienza con la etiqueta <li>
Los elementos de la lista estarán marcados con números de forma predeterminada.

Ejemplo:

<ol> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ol>

Listas de descripción

HTML también admite listas de descripciones.
Una lista de descripción es una lista de términos, con una descripción de cada término.

La etiquea <dl> define la lista de descripción, la etiqueta <dt> define el término (nombre)
y la etiqueta <dd> describe cata término.

Ejemplo:
Cafe
- Bebida negra caliente
Leche
- Bebida blanca fria

<dl> <dt>Cafe</dt> <dd>- Bebida negra caliente</dd> <dt>Leche</dt> <dd>- Bebida blanca fria</dd> </dl>

Tag descripcion
<ul> Lista desordenada
<ol> Lista ordenada
<li> define un item de la lista
<dl> define una descripcion de la lista
<dt> define un termino en una lista de descripcion
<dd> describe el termino en una lista de descripcion

Listas desordenadas

La etiqueta <ul> define una lista desordenada (con viñetas).

Una lista desordenada comienza con la etiqueta <ul>. Cada elemento de la lista comienza con la etiqueta <li>.
Los elementos de la lista estaran marcados con viñetas (pequeños circulos negros) de forma predeterminada:

Ejemplo:

<ul> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ul>

Lista desordenada: elija el marcador de elemento de lista

La propiedad CSS list-style-type se utiliza para definir el estilo del marcador de elementos de la lista.
Puede tener uno de los siguientes valores:

Valor descripción
disc Estable el marcador del elemento de la lista en una viñeta (predeterminado)
circle Estable el marcador del elemento de la lista en un circulo
square Estable el marcador del elemento de la lista en un cuadrado
none Los elementos de la lista no se marcaran

Listas anidadas

Las listas se pueden anidar (lista dentro de lista).

Ejemplo:
  • Cafe
  • Te
    • Te negro
    • Te verde
  • Leche

<ul> <li>Coffee</li> <li>Tea <ul> <li>Black tea</li> <li>Green tea</li> </ul> </li> <li>Milk</li> </ul>

Nota: Un elemento de lista (<li>) puede contener una nueva lista y otros elementos HTML como imágenes, enlaces, etc.

Lista horizontal con CSS

Las listas HTML se pueden diseñar de muchas maneras diferentes con CSS.

Una forma popular es diseñar una lista horizontalmente para crear un menú de navegación:

  • Home
  • News
  • Contact
  • About

<!DOCTYPE html> <html> <head> <style> ul { list-style-type: none; margin: 0; padding: 0; overflow: hidden; background-color: #333333; } li { float: left; } li a { display: block; color: white; text-align: center; padding: 16px; text-decoration: none; } li a:hover { background-color: #111111; } </style> </head> <body> <ul> <li><a href="#home">Home</a></li> <li><a href="#news">News</a></li> <li><a href="#contact">Contact</a></li> <li><a href="#about">About</a></li> </ul> </body> </html>

  • Utilice el elemento HTML <ul> para definir una lista desordenada
  • Utilice la propiedad CSS list-style-type para definir el marcador del elemento de la lista
  • Utilice el elemento HTML <li> para definir un elemento de lista
  • Las listas se pueden anidar
  • Los items de la lista pueden contener otros elementos HTML
  • Utilice la propiedad CSS float:left para mostrar la lista horizontalmente

Listas ordenadas

La etiqueta HTML <ol> define una lista ordenada.
Una lista ordenada puede ser numérica o alfabética.

Una lista ordenada comienza con la etiqueta <ol>. Cada elemento de la lista comienza con la etiqueta <li>
Los elementos de la lista estarán marcados con números de forma predeterminada.

Ejemplo:

<ol> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ol>

El atributo Type

El atributo type de la etiqueta <ol> define el tipo de marcador de elemento de la lista:

Type descripcion
type="1" Números (predeterminado)
type="A" Letras Mayúsculas
type:"a" Letras Minúsculas
type="I" Números Romanos en Mayúscula
type="i" Números Romanos en
Minúscula

Conteo de lista de control

De forma predeterminada, una lista ordenada comenzará a contar desde 1.
Si desea comenzar a contar desde un número específico, puede usar el atributo start=

Ejemplo:
  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk

<ol start="50"> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ol> <ol type="I" start="50"> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ol>
