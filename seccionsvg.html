Gráficos HTML SVG

SVG define gráficos basados en vectores en formato XML.

¿Qué es SVG?

  • SVG significa gráficos vectoriales escalables
  • SVG se utiliza para definir gráficos para la web
  • SVG es una recomendación del W3C

<svg>

El elemento <svg> es un contenedor para gráficos SVG.

SVG tiene varios métodos para dibujar trazados, cuadros, círculos, texto e imágenes gráficas.

Círculo SVG

<!DOCTYPE html>
<html>
<body>

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />
</svg>

</body>
</html>
  1. <svg width="100" height="100">
    • Abre un elemento SVG con una anchura de 100 unidades y una altura de 100 unidades.
  2. <circle cx="50" cy="50 r="40" stroke="green" stroke-width="4" fill="yellow" />
    • Dibuja un círculo en el lienzo SVG.
      • cx="50": La coordenada X del centro del círculo es 50 unidades.
      • cy="50": La coordenada Y del centro del círculo es 50 unidades.
      • r="40": El radio del círculo es de 40 unidades.
      • stroke="green": Establece el color del trazo (borde) del círculo a verde.
      • stroke-width="4": Establece el ancho del trazo del círculo a 4 unidades.
      • fill="yellow": Establece el color de relleno del círculo a amarillo.
  3. </svg>
    • Cierra el elemento SVG.

Este código SVG representa un círculo con un borde verde, un ancho de borde de 4 unidades y un relleno amarillo
en un lienzo de 100x100 unidades.

A diferencia de Canvas, SVG es una representación basada en XML de gráficos vectoriales
y permite una manipulación más fácil y flexible de los elementos gráficos.

Rectángulo SVG

<svg width="400" height="100">
  <rect width="400" height="100" style="fill:rgb(0,0,255);stroke-width:10;stroke:rgb(0,0,0)" />
</svg>
  1. <svg width="400" height="100">
    • Abre un elemento SVG con una anchura de 400 unidades y una altura de 100 unidades.
  2. <rect width="400" height="100" style="fill:rgb(0,0,255);stroke-width:10;stroke:rgb(0,0,0)" />
    • Dibuja un rectángulo en el lienzo SVG.
      • width="400": El ancho del rectángulo es de 400 unidades.
      • height="100": La altura del rectángulo es de 100 unidades.
      • style="fill:rgb(0,0,255);stroke-width:10;stroke:rgb(0,0,0)"
        Establece el estilo del rectángulo.
        • fill:rgb(0,0,255): Establece el color de relleno del rectángulo a azúl (rgb(0,0,255)).
        • stroke-width=10: Establece el ancho del trazo (borde) del rectángulo a 10 unidades.
        • stroke:rgb(0,0,0): Establece el color del trazo del rectángulo a negro (rgb(0,0,0))
  3. </svg>
    • Cierra el elemento SVG.

Este código SVG representa un rectángulo azul con un borde negro de 10 unidades de ancho en un lienzo de 400x100 unidades.

Así como en el ejemplo anterior, el código SVG permite la creación y manipulación de gráficos vectoriales de una manera
más declarativa y estructurada que Canvas, que se basa en píxeles y requiere más trabajo manual para realizar modificaciones.
