Elementos y técnicas de diseño HTML

Los sitios web suelen mostrar contenido en varias conlumnas (como una revista o un periódico)

Elementos de diseño HTML

HTML Posee varios elementos semánticos que definen las distintas partes de una página web:

  • <header> - Define el encabezado para un documento o sección
  • <nav> - Define un conjunto de enlaces de navegacion
  • <section> - Define una sección en un documento
  • <article> - Define un contenido independiente y autónomo
  • <aside> - Define contenido aparte del contenido (como una barra lateral)
  • <footer> - Define un pie de página para un documento o sección
  • <summary> - Define un encabezado para el elemento <details>
  • <details> - Define detalles adicionales que el usuario puede abrir y cerrar a eleccíon

Técnicas de diseño HTML

Existen cuatro técnicas diferentes para crear diseños de varias conlumnas.
Cada técnica tiene sus pros y sus contras:

  • Marco CSS (framework)
  • Propiedad flotante CSS (float property)
  • Caja flexible CSS (flexbox)
  • Cuadrícula CSS (grid)

Marcos CSS (Frameworks)

Los Marcos CSS, también llamados Frameworks, son conjuntos de estilos predefinidos
que simplifican el diseño web. Ofrecen ventajas como ahorro de tiempo consistencia y
capacidad de respuesta. Ejemplos populares incluten Bootstrap, Foundation y Materialize
Se utilizan descargando e integrando el archivo CSS en tu proyecto, personalizándolo
si es necesario y aplicando estilos a través de clases y componentes. Los Marcos CSS
facilitan el diseño web pero no son necesarios para todos los proyectos.

Diseño flotante CSS

Es común realizar diseños web completos utilizando la propiedad CSS float. Float es fácil de aprender :
solo necesita recordar cómo funcionan las propiedades float y clear.
Desventajas: los elementos flotantes están vinculados al flujo de documentos, lo que puede perjudicar la flexibilidad.

Diseño de caja flexible CSS (flexbox)

El uso de flexbox grarantiza que los elementos se comporten de manera predecible cuando el diseño
de la página debe adaptarse a diferentes tamaños de pantalla y difetentes dispositivos de visualización.

Diseño de cuadrícula CSS

El módulo CSS Grid Layout ofrece un sistema de diseño basado en cuadrículas, con filas y columnas
lo que facilita el diseño de páginas web sin tener que utilizar elementos flotantes ni posicionamiento.
