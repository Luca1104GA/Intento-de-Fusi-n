¿Qué son los Emojis?

Los emojis lucen como imagenes o iconos, pero no lo son.
Ellos son letras (caracteres) del conjunto de caracteres UTF-8 (Unicode)

UTF-8 Cubre casi todos los caracteres y símbolos del mundo.

El atributo charset

Para mostrar una página HTML correctamente, un navegador web debe conocer el juego de caracteres (charset) usado en la página.
Esto se especifica en la etiqueta <meta>.

<meta charset="UTF-8">

Si no se especifica, UTFC-8 es el juego de caracteres predeterminado en HTML.

Caracteres UTF-8

Muchos caracteres UTF-8 no se pueden escribir en un teclado, pero suempre se pueden mostrar
usando números (llamados números de entidad)

  • A es 65
  • B es 66
  • C es 67
Ejemplo
<!DOCTYPE html>
<html>
<meta charset="UTF-8">
<body>

<p>Esto mostrará A B C</p>
<p>Esto mostrará &#65; &#66; &#67;</p>

</body>
</html>

Ejemplo explicado

El elemento <meta charset="UTF-8"> define el set de caracteres.
Los caracteres A, B, y C, son mostrados por el numero 65, 66, y 67.

Para hacer que el navegador entienda que tu estas mostrando un caracter, debes comenzar el número de entidad con
&# y finalizar con ; (punto y coma).

Emojis

Los emojis también son caracteres del alfabeto UTF-8

  • 😄 es 128516
  • 😍 es 128525
  • 💗 es 128151
Ejemplo

Mi primer Emoji

😀

<!DOCTYPE html>
<html>
<meta charset="UTF-8">
<body>

<h1>Mi primer Emoji</h1>

<p>&#128512;</p>

</body>
</html>

Dado que los emojis son caracteres, se pueden copiar, mostrar, y cambiar de tamaño
como cualquier otro caracter en HTML.
